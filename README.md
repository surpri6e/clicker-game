# Clicker game

Утечек памяти нет. Никаких **dll** и **lib** библиотек не использовал.


Чтобы в Visual Studio 2022 сделать проект нужно выбрать пункт - 
***Пустой проект CLR (.NET Framework)***

 2. подсистема - *windows*
